# Fall25 Online Shopping

## Project Overview
This is a copy of the exact repository I utilized during my group project in my Database Management Class for our final project.
My group of four members constructed this database from scratch with varying degrees of contribution in different coding languages.
This project focuses on constructing a database from scratch for a small grocery store and pulling meaningful insights via queries.

Technologies Used
- Python
- SQL
- HTML
- CSS

## How to run this project - 
1. clone repository -
   ```
   git clone https://github.com/YOUR_USERNAME/Fall25OnlineShopping.git
   ```
2. install dependencies -
   ```
   pip install -r requirements.txt
   ```
3. run notebook or script
